#SharkTankBot

The souce code for @SharkTankBot on Twitter

To use:
Simply rename fakeconfig.js to config.js and plug in your Twitter credentials.

run `$ npm install`.

Then run 
```
node bot.js
```

Unless something went wrong, your bot will be up and retweeting!
